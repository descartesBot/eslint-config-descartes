## How to use?
- 1 - Install Eslint
```
npm instal eslint-config-descartes eslint --save-dev
yarn add eslint-config-descartes eslint --dev
```
- 2 - Configure Eslint
```json
{
  "extends": ["descartes"]
}
```
```json
{
  "extends": ["descartes/typescript"]
}
```
